# Glossopteris (Tongue Fern)

## Real-World Photography References

Fern frond photography. Focus on: tongue-shaped leaves with prominent central midrib, net-like venation pattern, woody trunk with rough bark.

## How to Use
1. Find high-quality macro/nature photographs matching the guidance above
2. Upload to Midjourney Discord
3. Copy the Discord CDN URL
4. Add to `sref_urls.json` under `"Glossopteris"`

## MJ Prompt Notes — Glossopteris

### CLIP-Optimized Shorthand
These comma-separated phrases are what the anatomy module injects into prompts:
```
  broad tongue-shaped leaves net-like venation
  thick woody deciduous canopy tree
  prominent central midvein on each leaf
  fallen tongue leaves covering ground
  25m Permian Gondwanan forest dominant
```

### Recommended --stylize Range
| Low | Default | High |
|-----|---------|------|
| 150 | **250** | 500 |

Use `--stylize 250` as a starting point. Lower values (150) preserve more anatomical accuracy. Higher values (500) allow more MJ artistic interpretation.

### Known MJ Failures
Watch for these common misrenders:
- ❌ fern — NOT a true fern; a seed fern with tongue-shaped LEAVES not fronds
- ❌ tropical palm — deciduous temperate-style tree, not tropical
