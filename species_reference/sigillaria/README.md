# Sigillaria (Seal Tree)

## Real-World Photography References

Palm trunk photography. Focus on: vertical rows of hexagonal leaf scars like pressed seals (NOT spiral like Lepidodendron), rarely branching, grass-like leaf tuft at top, bulbous base.

## How to Use
1. Find high-quality macro/nature photographs matching the guidance above
2. Upload to Midjourney Discord
3. Copy the Discord CDN URL
4. Add to `sref_urls.json` under `"Sigillaria"`

## MJ Prompt Notes — Sigillaria

### CLIP-Optimized Shorthand
These comma-separated phrases are what the anatomy module injects into prompts:
```
  hexagonal seal-like leaf scars in vertical rows
  tall straight rarely-branched column trunk
  grass-like leaf tuft at summit only
  smooth bark with raised round scars
  30m Carboniferous coal-swamp pillar tree
```

### Recommended --stylize Range
| Low | Default | High |
|-----|---------|------|
| 150 | **250** | 500 |

Use `--stylize 250` as a starting point. Lower values (150) preserve more anatomical accuracy. Higher values (500) allow more MJ artistic interpretation.

### Known MJ Failures
Watch for these common misrenders:
- ❌ palm tree — NOT a palm; Sigillaria has distinctive hexagonal ribbed bark
- ❌ broadleaf — leaves are long grass-like tufts, not broad leaves
