# Megalograptus (Clawed Sea Scorpion)

## Real-World Photography References

Scorpion + lobster photography. Focus on: elongated body, large spiny frontal claws, compound eyes, swimming paddles, segmented tail.

## How to Use
1. Find high-quality macro/nature photographs matching the guidance above
2. Upload to Midjourney Discord
3. Copy the Discord CDN URL
4. Add to `sref_urls.json` under `"Megalograptus"`

## MJ Prompt Notes — Megalograptus

### CLIP-Optimized Shorthand
These comma-separated phrases are what the anatomy module injects into prompts:
```
  massively elongated spiny chelicerae claws
  dark segmented eurypterid body
  swimming paddles on rear legs
  pointed telson tail spine
  1m Ordovician sea scorpion predator
```

### Recommended --stylize Range
| Low | Default | High |
|-----|---------|------|
| 100 | **150** | 300 |

Use `--stylize 150` as a starting point. Lower values (100) preserve more anatomical accuracy. Higher values (300) allow more MJ artistic interpretation.

### Known MJ Failures
Watch for these common misrenders:
- ❌ modern scorpion — eurypterid body plan, not curved-tail scorpion shape
- ❌ small — 1m predator, emphasize the massive spiny claws
