# Meganeura (Giant Dragonfly)

## Real-World Photography References

Modern dragonfly macro photography. Focus on: iridescent wing veins, compound eye facets, bristled leg segments, translucent wings with visible venation.

## How to Use
1. Find high-quality macro/nature photographs matching the guidance above
2. Upload to Midjourney Discord
3. Copy the Discord CDN URL
4. Add to `sref_urls.json` under `"Meganeura"`

## MJ Prompt Notes — Meganeura

### CLIP-Optimized Shorthand
These comma-separated phrases are what the anatomy module injects into prompts:
```
  eagle-sized giant dragonfly
  four transparent veined iridescent wings
  enormous compound eyes golf-ball sized
  glossy dark chitinous segmented body
  six bristled legs in prey-catching basket
  70cm wingspan largest flying insect ever
```

### Recommended --stylize Range
| Low | Default | High |
|-----|---------|------|
| 75 | **125** | 250 |

Use `--stylize 125` as a starting point. Lower values (75) preserve more anatomical accuracy. Higher values (250) allow more MJ artistic interpretation.

### Known MJ Failures
Watch for these common misrenders:
- ❌ tiny insect — must be eagle-SIZED, shot like a bird of prey not a macro insect
- ❌ butterfly wings — wings are transparent with dark veins, dragonfly-type
