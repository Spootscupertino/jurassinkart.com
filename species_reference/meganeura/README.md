# Meganeura (Giant Dragonfly)

## Real-World Photography References

Modern dragonfly macro photography. Focus on: iridescent wing veins, compound eye facets, bristled leg segments, translucent wings with visible venation.

## How to Use
1. Find high-quality macro/nature photographs matching the guidance above
2. Upload to Midjourney Discord
3. Copy the Discord CDN URL
4. Add to `sref_urls.json` under `"Meganeura"`
