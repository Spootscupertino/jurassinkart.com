## MJ Prompt Notes — Megalodon

### CLIP-Optimized Shorthand
These comma-separated phrases are what the anatomy module injects into prompts:
```
  hand-sized triangular serrated teeth
  broad heavy great-white body plan scaled up
  rough dermal-denticle sandpaper skin
  scarred blunt snout battle-worn
  powerful crescent caudal fin
  school-bus-sized 15m apex shark
```

### Recommended --stylize Range
| Low | Default | High |
|-----|---------|------|
| 100 | **175** | 350 |

Use `--stylize 175` as a starting point. Lower values (100) preserve more anatomical accuracy. Higher values (350) allow more MJ artistic interpretation.

### Known MJ Failures
Watch for these common misrenders:
- ❌ smooth skin — shark skin is rough sandpaper-like dermal denticles
- ❌ whale shape — body plan is great white shark scaled up, not whale-shaped
