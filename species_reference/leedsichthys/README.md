## MJ Prompt Notes — Leedsichthys

### CLIP-Optimized Shorthand
These comma-separated phrases are what the anatomy module injects into prompts:
```
  enormous gaping filter-feeding mouth
  whale-shark-like mottled dark skin
  massive barrel body paddle fins
  gill rakers for straining plankton
  gentle slow-cruising giant
  16m whale-sized bony fish
```

### Recommended --stylize Range
| Low | Default | High |
|-----|---------|------|
| 125 | **200** | 400 |

Use `--stylize 200` as a starting point. Lower values (125) preserve more anatomical accuracy. Higher values (400) allow more MJ artistic interpretation.

### Known MJ Failures
Watch for these common misrenders:
- ❌ shark shape — Leedsichthys is a bony fish, not a shark; different fin structure
- ❌ teeth — has gill rakers for filter feeding, no predatory teeth
