# Pulmonoscorpius (Giant Scorpion)

## Real-World Photography References

Emperor scorpion macro photography. Focus on: robust pedipalps, curved stinger, segmented metasoma tail, glossy dark exoskeleton under UV or natural light.

## How to Use
1. Find high-quality macro/nature photographs matching the guidance above
2. Upload to Midjourney Discord
3. Copy the Discord CDN URL
4. Add to `sref_urls.json` under `"Pulmonoscorpius"`
