# Arthropleura (Giant Millipede)

## Real-World Photography References

Modern giant millipede macro photography. Focus on: segmented body plates, paired leg rows, rounded head, dark chitinous exoskeleton with natural wear and scratches.

## How to Use
1. Find high-quality macro/nature photographs matching the guidance above
2. Upload to Midjourney Discord
3. Copy the Discord CDN URL
4. Add to `sref_urls.json` under `"Arthropleura"`

## MJ Prompt Notes — Arthropleura

### CLIP-Optimized Shorthand
These comma-separated phrases are what the anatomy module injects into prompts:
```
  car-length giant millipede
  dinner-plate-sized armored body segments
  dozens of thick paired legs
  dark chitinous interlocking plates
  2.6m longest land arthropod ever
  low ground-level dwarfing ferns
```

### Recommended --stylize Range
| Low | Default | High |
|-----|---------|------|
| 75 | **125** | 250 |

Use `--stylize 125` as a starting point. Lower values (75) preserve more anatomical accuracy. Higher values (250) allow more MJ artistic interpretation.

### Known MJ Failures
Watch for these common misrenders:
- ❌ small bug — must be car-length, shot like a large reptile not a small millipede
- ❌ centipede — Arthropleura is a MILLIPEDE (two leg pairs per segment), not centipede
