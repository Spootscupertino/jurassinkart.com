# Lepidodendron (Scale Tree)

## Real-World Photography References

Tree fern + palm trunk photography. Focus on: diamond-shaped leaf scars in spiral pattern covering entire trunk, Y-branching crown, grass-like leaf tufts at branch tips.

## How to Use
1. Find high-quality macro/nature photographs matching the guidance above
2. Upload to Midjourney Discord
3. Copy the Discord CDN URL
4. Add to `sref_urls.json` under `"Lepidodendron"`

## MJ Prompt Notes — Lepidodendron

### CLIP-Optimized Shorthand
These comma-separated phrases are what the anatomy module injects into prompts:
```
  diamond-shaped spiral leaf scars on trunk
  tall straight unbranched columnar trunk
  grass-like drooping leaf crown at top only
  reptile-scale-patterned bark texture
  40m towering Carboniferous club moss
```

### Recommended --stylize Range
| Low | Default | High |
|-----|---------|------|
| 150 | **250** | 500 |

Use `--stylize 250` as a starting point. Lower values (150) preserve more anatomical accuracy. Higher values (500) allow more MJ artistic interpretation.

### Known MJ Failures
Watch for these common misrenders:
- ❌ palm tree — NO palms; lepidodendron is a scale tree with diamond-scarred bark
- ❌ oak or broadleaf — leaves are long grass-like, NOT broad leaves
