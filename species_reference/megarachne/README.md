# Megarachne (Giant Eurypterid)

## Real-World Photography References

Eurypterid body plan reference (NOT spider). Focus on: broad oval body, strong walking legs, small chelicerae, compound eyes, dark segmented exoskeleton.

## How to Use
1. Find high-quality macro/nature photographs matching the guidance above
2. Upload to Midjourney Discord
3. Copy the Discord CDN URL
4. Add to `sref_urls.json` under `"Megarachne"`
