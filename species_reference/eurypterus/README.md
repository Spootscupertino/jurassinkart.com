# Eurypterus (Sea Scorpion)

## Real-World Photography References

Horseshoe crab photography. Focus on: streamlined body, paddle-shaped rear legs, telson tail spike, olive-brown smooth exoskeleton.

## How to Use
1. Find high-quality macro/nature photographs matching the guidance above
2. Upload to Midjourney Discord
3. Copy the Discord CDN URL
4. Add to `sref_urls.json` under `"Eurypterus"`

## MJ Prompt Notes — Eurypterus

### CLIP-Optimized Shorthand
These comma-separated phrases are what the anatomy module injects into prompts:
```
  compact olive-brown sea scorpion
  paddle-shaped rear swimming legs
  streamlined oval segmented body
  tail spike telson for defense
  horseshoe-crab-sized 30cm eurypterid
```

### Recommended --stylize Range
| Low | Default | High |
|-----|---------|------|
| 100 | **150** | 300 |

Use `--stylize 150` as a starting point. Lower values (100) preserve more anatomical accuracy. Higher values (300) allow more MJ artistic interpretation.

### Known MJ Failures
Watch for these common misrenders:
- ❌ modern scorpion — flat eurypterid body plan, not curved-tail scorpion
- ❌ too large — Eurypterus is relatively small (30cm), horseshoe-crab-sized
