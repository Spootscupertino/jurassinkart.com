# Williamsonia (Bennettite)

## Real-World Photography References

Cycad + magnolia photography. Focus on: cycad-like trunk, large flower-like reproductive structures, crown of pinnate fronds, stout columnar stem with leaf-base armor.

## How to Use
1. Find high-quality macro/nature photographs matching the guidance above
2. Upload to Midjourney Discord
3. Copy the Discord CDN URL
4. Add to `sref_urls.json` under `"Williamsonia"`

## MJ Prompt Notes — Williamsonia

### CLIP-Optimized Shorthand
These comma-separated phrases are what the anatomy module injects into prompts:
```
  flower-like cones between palm fronds
  short stocky trunk with leaf-base armor
  crown of pinnate feather-like fronds
  cycad-like palm-like overall form
  3m stocky Mesozoic bennettitale
```

### Recommended --stylize Range
| Low | Default | High |
|-----|---------|------|
| 150 | **250** | 500 |

Use `--stylize 250` as a starting point. Lower values (150) preserve more anatomical accuracy. Higher values (500) allow more MJ artistic interpretation.

### Known MJ Failures
Watch for these common misrenders:
- ❌ palm tree — NOT a palm; bennettitalean with stout trunk and flower-like structures
- ❌ modern cycad — similar but with large flower-like reproductive organs unlike cycads
