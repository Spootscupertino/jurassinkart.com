# Jaekelopterus (Giant Sea Scorpion)

## Real-World Photography References

Horseshoe crab + scorpion photography. Focus on: broad flat body, large chelicerae pincers, compound eyes, paddle-shaped swimming legs, dark exoskeleton.

## How to Use
1. Find high-quality macro/nature photographs matching the guidance above
2. Upload to Midjourney Discord
3. Copy the Discord CDN URL
4. Add to `sref_urls.json` under `"Jaekelopterus"`
