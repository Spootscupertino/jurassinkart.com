# Jaekelopterus (Giant Sea Scorpion)

## Real-World Photography References

Horseshoe crab + scorpion photography. Focus on: broad flat body, large chelicerae pincers, compound eyes, paddle-shaped swimming legs, dark exoskeleton.

## How to Use
1. Find high-quality macro/nature photographs matching the guidance above
2. Upload to Midjourney Discord
3. Copy the Discord CDN URL
4. Add to `sref_urls.json` under `"Jaekelopterus"`

## MJ Prompt Notes — Jaekelopterus

### CLIP-Optimized Shorthand
These comma-separated phrases are what the anatomy module injects into prompts:
```
  massive serrated chelicerae pincers
  crocodile-sized sea scorpion
  broad flat segmented body
  rear paddle-shaped swimming legs
  dark scarred chitinous exoskeleton
  2.5m largest arthropod predator ever
```

### Recommended --stylize Range
| Low | Default | High |
|-----|---------|------|
| 75 | **125** | 250 |

Use `--stylize 125` as a starting point. Lower values (75) preserve more anatomical accuracy. Higher values (250) allow more MJ artistic interpretation.

### Known MJ Failures
Watch for these common misrenders:
- ❌ small scorpion — must be crocodile-SIZED, shot like an alligator-sized predator
- ❌ modern scorpion shape — eurypterid body plan is FLAT and broad, not curved-tail scorpion
