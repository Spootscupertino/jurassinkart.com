# Archaefructus (First Flower)

## Real-World Photography References

Water lily + simple wildflower photography. Focus on: slender aquatic stems, small simple flowers without showy petals, dissected leaves, delicate branching in shallow water.

## How to Use
1. Find high-quality macro/nature photographs matching the guidance above
2. Upload to Midjourney Discord
3. Copy the Discord CDN URL
4. Add to `sref_urls.json` under `"Archaefructus"`

## MJ Prompt Notes — Archaefructus

### CLIP-Optimized Shorthand
These comma-separated phrases are what the anatomy module injects into prompts:
```
  tiny primitive flower-like structures at tips
  slender herbaceous aquatic stems
  finely dissected feathery compound leaves
  rooted in shallow lake sediment
  50cm earliest known flowering plant
```

### Recommended --stylize Range
| Low | Default | High |
|-----|---------|------|
| 175 | **300** | 600 |

Use `--stylize 300` as a starting point. Lower values (175) preserve more anatomical accuracy. Higher values (600) allow more MJ artistic interpretation.

### Known MJ Failures
Watch for these common misrenders:
- ❌ large tree — tiny aquatic herb (50cm), grows IN water not on land
- ❌ showy flowers — flowers are tiny and primitive, not large colorful blooms
