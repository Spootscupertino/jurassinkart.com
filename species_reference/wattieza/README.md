# Wattieza (First Tree)

## Real-World Photography References

Tree fern photography. Focus on: frond-scarred trunk like a tree fern, crown of branch-like fronds, no true leaves, primitive tree structure. Oldest known tree ~385 MYA.

## How to Use
1. Find high-quality macro/nature photographs matching the guidance above
2. Upload to Midjourney Discord
3. Copy the Discord CDN URL
4. Add to `sref_urls.json` under `"Wattieza"`

## MJ Prompt Notes — Wattieza

```

  tall slender unbranched columnar trunk
```

### Recommended --stylize Range
| Low | Default | High |
|-----|---------|------|
| 175 | **300** | 600 |

Use `--stylize 300` as a starting point. Lower values (175) preserve more anatomical accuracy. Higher values (600) allow more MJ artistic interpretation.

### Known MJ Failures
Watch for these common misrenders:
- ❌ modern tree fern — Wattieza is simpler, more primitive than modern tree ferns
- ❌ branching tree — BRANCHLESS single trunk with crown of fronds
