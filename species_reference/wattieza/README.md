# Wattieza (First Tree)

## Real-World Photography References

Tree fern photography. Focus on: frond-scarred trunk like a tree fern, crown of branch-like fronds, no true leaves, primitive tree structure. Oldest known tree ~385 MYA.

## How to Use
1. Find high-quality macro/nature photographs matching the guidance above
2. Upload to Midjourney Discord
3. Copy the Discord CDN URL
4. Add to `sref_urls.json` under `"Wattieza"`
