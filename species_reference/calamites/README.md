# Calamites (Giant Horsetail)

## Real-World Photography References

Modern horsetail + bamboo photography (scaled up). Focus on: jointed segmented trunk with vertical ribbing, whorls of needle-like leaves at each node, cone-shaped strobili.

## How to Use
1. Find high-quality macro/nature photographs matching the guidance above
2. Upload to Midjourney Discord
3. Copy the Discord CDN URL
4. Add to `sref_urls.json` under `"Calamites"`

## MJ Prompt Notes — Calamites

### CLIP-Optimized Shorthand
These comma-separated phrases are what the anatomy module injects into prompts:
```
  jointed bamboo-like segmented trunk
  needle-branch whorls at each node
  ribbed longitudinal trunk surface
  feathery tiered canopy silhouette
  20m giant Carboniferous horsetail
```

### Recommended --stylize Range
| Low | Default | High |
|-----|---------|------|
| 150 | **250** | 500 |

Use `--stylize 250` as a starting point. Lower values (150) preserve more anatomical accuracy. Higher values (500) allow more MJ artistic interpretation.

### Known MJ Failures
Watch for these common misrenders:
- ❌ bamboo — similar but NOT bamboo; Calamites is a giant horsetail with distinct nodes
- ❌ modern horsetail — must be TREE-sized (20m), not small modern horsetail
