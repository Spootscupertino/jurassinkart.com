# Anomalocaris (Cambrian Predator)

## Real-World Photography References

Mantis shrimp + cuttlefish photography. Focus on: flattened oval body, lateral swimming lobes, stalked compound eyes, grasping frontal appendages, translucent body.

## How to Use
1. Find high-quality macro/nature photographs matching the guidance above
2. Upload to Midjourney Discord
3. Copy the Discord CDN URL
4. Add to `sref_urls.json` under `"Anomalocaris"`

## MJ Prompt Notes — Anomalocaris

### CLIP-Optimized Shorthand
These comma-separated phrases are what the anatomy module injects into prompts:
```
  two large spiny grasping frontal appendages
  circular pineapple-ring mouth
  stalked compound eyes 16000 lenses
  rippling lateral swimming lobes
  semi-translucent flattened oval body
  1m Cambrian apex predator
```

### Recommended --stylize Range
| Low | Default | High |
|-----|---------|------|
| 100 | **175** | 350 |

Use `--stylize 175` as a starting point. Lower values (100) preserve more anatomical accuracy. Higher values (350) allow more MJ artistic interpretation.

### Known MJ Failures
Watch for these common misrenders:
- ❌ fish shape — flat oval body with lateral lobes, nothing like a fish
- ❌ hard shell — semi-translucent, NOT hard-shelled like a crab
- ❌ tiny — must be 1m, the apex predator of the Cambrian
